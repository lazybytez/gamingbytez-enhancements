# Mythic Altar — Weather Rituals

These recipes change the weather in the current world. Each ritual broadcasts a message to all players on completion. If the target weather is already active, the ingredients are returned and no effect is applied.

See [overview.md](overview.md) for how to build and use the altar.

---

## Sun Ritual

Clears the weather (stops rain and thunderstorms).

### Ingredients

| Pedestal | Item |
|---|---|
| Center | Lightning Rod |
| North-West | Sponge |
| North-East | Sponge |
| South-West | Sponge |
| South-East | Sponge |

### Result

- Weather is set to clear.
- A **Wet Sponge** drops from each of the four outer pedestals.
- The center Lightning Rod is consumed.
- Yellow particle lines animate from the outer pedestals to the center.

---

## Rain Ritual

Starts rain (non-thundering storm).

### Ingredients

| Pedestal | Item |
|---|---|
| Center | Water Bucket |
| North-West | Water Bucket |
| North-East | Water Bucket |
| South-West | Water Bucket |
| South-East | Water Bucket |

### Result

- Weather is set to rain (storm without thunder).
- An empty **Bucket** drops from all five pedestals.
- Blue particle lines animate from the outer pedestals to the center.

---

### Ingredients

| Pedestal | Item |
|---|---|

| North-West | Water Bucket |
| North-East | Water Bucket |
| South-West | Water Bucket |
